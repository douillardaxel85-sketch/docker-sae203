SAE 2.03 : Installation de Services Réseaux

Service de Partage de Fichiers (Samba) sous Docker

1. Introduction

Ce projet consiste à déployer un serveur de fichiers sécurisé en utilisant la technologie de conteneurisation Docker. Nous avons choisi le service Samba pour sa compatibilité universelle et sa gestion fine des droits utilisateurs.

2. Choix Techniques

  • Base : Debian Latest (Contrainte de la SAE)
  • Service : Samba (smbd)
  • Ports : 445 (SMB) mappé sur 4445 pour éviter les conflits Windows.

3. Preuve de fonctionnement

Le serveur a été vérifié à l'aide de la commande ps aux montrant les processus actifs :

USER  PID %CPU %MEM  COMMAND
root    1  0.0  0.1  smbd --foreground --no-process-group
